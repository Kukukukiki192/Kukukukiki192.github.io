注:

sudo cd root 并没有切换到root目录

sudo 是一种程序，用于提升用户的权限

shell是一个命令解析器，sudo cd是错误的，因为cd是shell内置的，不是系统里面的

sudo可以运行系统带的命令，但无法运行系统一个软件中的命令

所以这里要切换到root目录，只能先切换到root用户
